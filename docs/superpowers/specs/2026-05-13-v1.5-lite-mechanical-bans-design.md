5 bài hôm nay xuất hiện 4 patterns lỗi:

### Pattern A — Bình dân giả (verb tự chế)
AI generalize STYLE từ V1.2/V1.3 DRAMATIC_VERBS list, không pile-on trực tiếp:
- "chấm đích" (FPT title+body, MSN body) — NOT in DRAMATIC_VERBS list
- "vọt lãi" (MSN title) — NOT explicit (chỉ "vọt" + "ăn lãi" gốc)
- "xén lợi FPT mẹ" — NOT explicit (chỉ "xén cổ tức" gốc)
- "VCBS chấm 111.421 đồng" — "chấm" không phải verb định giá
- "đẻ ra tỉnh thứ tư" — extended từ "đẻ ra"
- "Playbook lặp từ" — NOT in list at all

### Pattern B+C — Hán-Việt formal + abbreviation không expand
- "độc bản" / "chưa hội đủ" / "tái định giá" / "phương án xử lý" / "ban hành nghị quyết"
- "BCA" / "GRDP" / "SCIC" — không expand on first mention

### Pattern D — Fabricated data
- FPT bài closing: "tích lũy dưới 145 nghìn/cp" — FPT thực tế ~70 nghìn. Master hallucinated price target.

### Pattern E — Length squeeze
V1.3 -20% length cap → Master xé câu thành tail fragments.

### Root cause (verified via advisor)

User directive: **"Định nghĩa thôi còn để AI linh động chứ — pipeline ép dùng list verb thì xem xét lại"**

Advisor check 1 (grep verified): bad verbs ("chấm đích / vọt lãi / xén lợi") **KHÔNG có trong DRAMATIC_VERBS list**. AI generalize STYLE từ list examples + Master prompt examples. Dropping lists alone không fix Pattern A. Fix gốc = Master prompt revision với "DO NOT invent" examples.

Advisor concern 3: V1.5 LLM judge = ADD enforcement gate. User intent "AI linh động" có thể mean **less gates** (Approach 1 trust-AI), not MORE (Approach 3 LLM judge).

Advisor concern 4: 3rd iteration in 12 hours, scope growing. Simpler 1-commit experiment likely fixes 80%.

## Voice intent (master prompt — definition-only)

```
## Voice — bình dân xuồng xã nguy hiểm (V1.5-lite definition)

Reader is everyday Vietnamese investor (NĐT). Body must read naturally —
NOT thông cáo báo chí, NOT AI-generated.

### Apply intent (NOT word list)

- USE concrete Vietnamese verbs THAT FIT the action — pick from natural
  Vietnamese vocabulary, NOT from a prescribed list. Examples ok ngữ cảnh:
  "ăn lãi" (lãi tăng), "khoe lãi" (announce earnings), "dồn vốn" (concentrate
  capital), "bơm vốn" (inject capital), "cắt giảm" (reduce headcount),
  "sa thải" (lay off). Pick per fit, not per list.

- DO NOT invent verb-noun combos not in natural Vietnamese:
  - ❌ "chấm đích" — not Vietnamese. Use "nhắm tới / chọn / chấm mốc"
  - ❌ "vọt lãi" — odd combo. Use "lãi vọt / lãi tăng vọt"
  - ❌ "xén lợi FPT mẹ" — kệch cỡn. Use "ảnh hưởng lợi nhuận FPT mẹ"
  - ❌ "VCBS chấm 111.421 đồng" — "chấm" = mark/dot. Use "VCBS định giá 111.421 đồng"
  - ❌ "đẻ ra tỉnh thứ tư" — forced metaphor. Use "mở rộng sang tỉnh thứ tư"
  - ❌ "Playbook lặp từ" — English word + odd verb. Use "lặp lại mô hình từ"
  - If reader chưa từng nghe câu đó trên báo / đời sống, REWRITE.

### AVOID Hán-Việt formal — use bình dân equivalent

| Hán-Việt formal | Bình dân |
|---|---|
| độc bản | duy nhất / chỉ có 1 |
| hội đủ | đủ |
| tái định giá | định giá lại |
| cấu trúc vốn | cơ cấu vốn |
| phương án xử lý | cách xử lý |
| triển khai | làm / thực hiện |
| ban hành | ra / công bố |
| thông qua nghị quyết | quyết / chốt |
| dự kiến đạt | nhắm tới |
| hoàn thành kế hoạch | đạt kế hoạch |
| khái niệm | ý tưởng |
| hệ tư tưởng | tư tưởng |
| ngưỡng yêu cầu | mức tối thiểu |
| chế tài | chế độ phạt |

### EXPAND abbreviations first occurrence

- "Bộ Công An (BCA)" lần đầu, sau đó BCA
- "Tổng sản phẩm địa bàn (GRDP)" lần đầu, sau đó GRDP
- "Tổng công ty Đầu tư và Kinh doanh vốn Nhà nước (SCIC)" lần đầu
- Allowlist (không cần expand — phổ biến): ESOP / NIM / ROE / ROA / EPS /
  IPO / CASA / NPL / LNTT / LNST / Q1 / Q2 / Q3 / Q4

### VERIFY current price from Finpath API (mandatory step 0.5)

Trước khi quote price target trong closing:
1. Call `lib.finpath_api.get_stock_price(ticker)` → current price
2. Target price MUST be within ±50% current
3. NEVER fabricate price target. Log actual current in pipeline_log.

### Metaphor optional, not mandatory

Use khi natural ("gấp 3 lần Vietcombank cùng kỳ", "ngang ngửa BIDV"), drop khi cưỡng ép ("đẻ ra tỉnh thứ tư" = forced).
```

## Abbreviation expand check

`check_abbreviation_expanded(body)`:
1. Strip Skeptic + pipeline_log
2. Find all 3-4 uppercase letter sequences (`\b[A-Z]{3,4}\b`)
3. Cho mỗi unique abbreviation:
   - Nếu trong `NATURALIZED_FINANCE_TERMS` allowlist → skip
   - Nếu là ticker (in finpath sectors cache) → skip
   - Nếu first occurrence followed by `(<text>)` → pass (expanded)
   - Ngược lại → fail with abbreviation name
4. Return `{pass, missing_expansions: [...]}`

NATURALIZED_FINANCE_TERMS (already exists) covers ESOP/NIM/ROE/CASA/NPL/etc.

---

## Price realistic check

`check_price_realistic(body, ticker)`:
1. Strip body, find closing paragraph (last block)
2. Extract numbers preceded by stock-price units: "nghìn/cp", "nghìn đồng", "nghìn", `\d+\.\d{3}\s*đồng/cp`
3. Fetch Finpath current price via `lib.finpath_api.get_stock_price(ticker)`
4. For each price target found, assert within ±50% current
5. Return `{pass, current_price, targets_found, out_of_range}`

If Finpath API unavailable, gate degrades to pass with warning (don't block pipeline).
